## Projeto de Análise de Dados de Vendas

Este projeto realiza uma análise de dados de vendas de produtos em diferentes lojas, comparando o faturamento obtido com as metas estabelecidas para cada gerente.

A análise foi desenvolvida com Python e Pandas, permitindo organizar, tratar e interpretar os dados para identificar o desempenho de cada loja, verificar o cumprimento das metas e gerar insights estratégicos para tomada de decisão.

### Objetivos do projeto
- Comparar o faturamento das lojas com as metas definidas
- Avaliar o desempenho de cada gerente
- Identificar lojas com melhor desempenho
- Gerar insights a partir dos dados de vendas
- Praticar análise e manipulação de dados com Python

### Tecnologias utilizadas
- Python
- Pandas
- NumPy
